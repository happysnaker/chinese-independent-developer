# Security Policy

This repository is mostly a curated public list, but please report privately if you find exposed secrets, private screenshots, personal data, or other sensitive information in the public content.

Email: `happysnaker@foxmail.com`

Suggested subject:

```text
[chinese-independent-developer security] short issue summary
```

Please include the affected file/section and a short explanation. Do not paste sensitive data into a public issue.
